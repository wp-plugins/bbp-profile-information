For bbPress - adds up to 4 fields to the bbp user profile and displays any combination of these under the authors avatar in topics and replies

== Description ==
This plugin for bbpress adds up to 4 fields to the bbp user profile.

The web administrator sets up the field names, and the forum users populate their details.
<ul>
	<li>Many sites would like to have the forum user's location, so city and state, or country can be set up as fields</li>
	<li>On a gaming site you could fields like favourite game, platform, level achieved, highest score, favourite character.</li>
	<li>On a diet site you could let users have target weight, diet plan, weight lost so far etc.</li>
	<li>On a film site you could have favourite film, best actor, film genre.</li>
	<li>Or on a golf site you could have handicap.</li>
</ul>

In fact up to 4 pieces of information that your forum users might want to display.


Works with bbpress 2.5.3




== Installation ==
1. Go to Dashboard>plugins>add new
2. Search for 'bbp profile information'
3. Click install
4. and then activate
5. Go to Dashboard>settings>bbp profile information and follow the instructions to select which items to display, and what labels they should have
